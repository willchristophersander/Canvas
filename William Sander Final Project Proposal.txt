Canvas is a C++ drawing and graphics application using OpenGL that allows users to draw on a digital canvas with multiple tools, layers, and real-time interaction. It will include pencil, shape, fill, eraser, and spray tools, along with a color picker, adjustable brush sizes, and graphical UI controls. Canvas will support multiple layers with opacity and visibility controls, file import/export (PNG/JPEG), and a full undo/redo system using a command-based architecture. The project will also incorporate a Python-based filter pipeline, enabling advanced image processing such as blur, edge detection, or sharpening. The goal is to demonstrate mastery of event-driven programming, object-oriented design, C++ graphics, debugging, and multi-language integration.
